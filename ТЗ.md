Проект простого бота для перевода аудио в текст с помощью api openai

Бот должен принимать аудио (или несколько) в виде голосовых сообщений (.ogg) или аудиофайлов и в ответ присылать текстовую транскрибацию.

Модель должна настраиваться через переменную среды, по умолчанию 4o mini transcribe (или как-то так)

Бот должен переводить формат аудио, если присланный не принимается api openai

Доступ только пользователям, айди которых есть в ALLOWED_IDS переменной, все остальные получают "доступ запрещен"

Нужно не забыть отредактировать йамл и Докерфайл, а также имя пакета. Порт 2010, название сервиса и пакета sttbot, домен sttbot.tgbots.site
